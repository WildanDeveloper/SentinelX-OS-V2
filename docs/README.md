# SentinelX OS V2

SentinelX OS is a next-generation hybrid operating system, combining the rolling-release model of Arch Linux with the stability and enterprise-level features of Red Hat.

## Key Features
- **Hybrid Architecture**: Rolling-release updates from Arch Linux merged with Red Hat's management system.
- **Custom Kernel**: Built and optimized for performance.
- **Advanced Security**: AppArmor and SELinux for multi-layer protection.
- **Atomic Updates**: Reliable system upgrades with auto-rollback capability.

## Repository Structure
- `kernel/`: Custom kernel development.
- `drivers/`: Device drivers for hardware compatibility.
- `tools/`: Scripts for building, debugging, and utilities.
- `configs/`: System and kernel configuration files.
- `packages/`: Hybrid integration with Arch and RPM packages.
- `init/`: Bootstrapping and system initialization setups.
